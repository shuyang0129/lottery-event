## 關於

- 使用者可以在指定抽獎次數內抽獎
- 兩次抽獎需要間隔五秒
- 得獎獎項由 API 決定
- 點擊抽獎時，會先檢視使用者是否為登入狀態，或是 Token 是否失效，並顯示對應邏輯
